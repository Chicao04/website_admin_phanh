# HƯỚNG DẪN CHẠY DỰ ÁN
## 1. Backend (lms-backend)
-   Cài thư viện:
        cd lms-backend
        npm install
-   Chạy dự án:
        npm start
## 2. Frontend (lms-frontend)
-   Cài thư viện:
        cd lms-frontend
        npm install
-   Chạy dự án:
        npm run dev
## 3. Ghi chú
-   Backend chạy tại: http://localhost:8080\
-   Frontend chạy tại: http://localhost:5173
